
# 📝 To-Do List Web App (Flask)

A simple and efficient task management web application built using Python Flask. This project allows users to manage daily tasks with a clean and user-friendly interface.

---

## 🚀 Features
- ➕ Add new tasks
- ❌ Delete tasks
- ⚡ Fast and lightweight
- 🎨 Clean and responsive UI

---

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS
- **Templating:** Jinja2

---

## 📂 Project Structure
todo-flask-app
│
├── app.py
└── templates
└── index.html

---

## ▶️ How to Run Locally

1. Install Flask:
2. py -m pip install flask


2. Run the application:

py app.py


3. Open in browser:

http://127.0.0.1:5000/
